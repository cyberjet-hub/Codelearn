HTML Essentials

A quick summary of web structure before your final assessment.

01. Element Anatomy

HTML elements are the building blocks of pages. Most consist of an opening tag, content, and a closing tag.

<p>This is a paragraph.</p>
  • The closing tag always has a forward slash /.
  • Some tags like <img> and <br> are "Self-closing."

02. Boilerplate Structure

Every HTML file follows a specific hierarchy to be valid. The <body> contains everything visible to the user.

<!DOCTYPE html> <html> <head> <title>Page Title</title> </head> <body> <!-- Visible content goes here --> </body> </html>

03. Semantic Elements

  • <h1> to <h6>: Define headings (H1 is the most important).
  • <p>: Defines a text paragraph.
  • <a href="...">: Creates a hyperlink to another page.
  • <img src="...">: Embeds an image into the document.
