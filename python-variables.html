Python Variables

A variable is a reserved memory location to store values. In Python, you don't need to declare the type—it is determined automatically.

String (str) Text

Encased in quotes: name = "Zion"

Integer (int) Whole Number

No decimals: age = 16

Float (float) Decimal

Numbers with dots: score = 98.5
