Variables and Types

In the world of programming, a variable is a reserved memory location to store values. This means when you create a variable, you reserve some space in memory. Based on the data type of a variable, the interpreter allocates memory and decides what can be stored in the reserved memory.

PYTHON 3.x
# Assigning values to variables
username = "cyber_ninja"
points = 1250
is_online = True

# Printing the stored data
print(f"User: {username} | Points: {points}")

Unlike other languages like C++ or Java, Python has no command for declaring a variable. A variable is created the moment you first assign a value to it. This makes it incredibly flexible for rapid development.

The Four Pillars of Data

  • Integers (int): Whole numbers, positive or negative (e.g., 10, -5).
  • Floats: Numbers containing a decimal point (e.g., 3.14, 2.0).
  • Strings (str): Sequence of characters wrapped in quotes (e.g., "Hello").
  • Booleans (bool): Represents one of two values: True or False.
