Networks & The Internet

A technical summary of data transmission, protocols, and the global infrastructure of the web.

01 Network Types

Networks are categorized by their scale and geographical reach.

PAN (Personal)
LAN (Local - School/Home)
WAN (Wide - The Internet)
  • LAN: High speed, low cost, small area.
  • WAN: Connects LANs using satellites or fiber cables.

02 Data Packets

Data is broken into "packets" before being sent. Each packet contains the payload and a header.

  • IP Address: The destination "house number."
  • MAC Address: The unique "serial number" of the hardware.
  • Routers: The "traffic police" directing packets.

03 The Language of the Web (Protocols)

Protocols are sets of rules that govern how devices communicate.

HTTP/S (Web Pages) FTP (File Transfer) SMTP (Email) DNS (Domain Name System) TCP/UDP (Transmission)
Pro-Tip: DNS is like a phonebook. It turns human names (google.com) into machine numbers (142.250.190.46).

04 Essential Hardware

  • NIC: Network Interface Card (Required for connection).
  • Switch: Connects devices within a single LAN.
  • Modem: Converts signals from the ISP for your router.
  • Fiber Optics: Uses light pulses to transmit data at 300,000 km/s.

05 Network Security

  • Firewall: Monitors incoming/outgoing traffic.
  • Encryption: Scrambling data so only the receiver can read it.
  • VPN: Creates a "secure tunnel" over a public network.
